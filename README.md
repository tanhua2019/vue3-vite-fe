#### getBoundingClientRect() 
- top：元素上边到视窗上边的距离;
- right：元素右边到视窗左边的距离;
- bottom：元素下边到视窗上边的距离;
- left：元素左边到视窗左边的距离;
- width：是元素自身的宽
- height 是元素自身的高
